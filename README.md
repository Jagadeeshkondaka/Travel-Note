🌍 Travel Note

A full-stack travel journaling web application that allows users to document, organize, and relive their travel experiences. Users can create, edit, and manage travel notes with images and descriptions.

🚀 Features
✍️ Create and manage travel notes
🖼️ Upload and display images for each trip
📍 Add location details to travel entries
✏️ Edit and delete notes
🔐 User authentication (Signup/Login)
📱 Responsive design for all devices
